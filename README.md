

# Velocity Admin UI Template

## Project Overview

This project is a modern, open-source UI template built with React and Tailwind CSS, designed for dashboards, admin panels, and SaaS applications. It aims to provide a clean, professional, and highly customizable foundation for new web applications.

## Key Features

- **Responsive Design:** Fully fluid layout adapting to desktops, tablets, and mobile devices.
- **Dark/Light Mode:** Integrated theme switcher for user preference.
- **Reusable Components:** A set of essential UI components (Buttons, Cards, Forms, Modals, Navigation).
- **Example Pages:** Pre-built pages to showcase functionality (Dashboard, Profile/Settings, Login).

## Technology Stack

- **Framework:** React
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Icons:** Heroicons
- **Fonts:** Google fonts

## Getting Started

To get started with development:

1. **Install dependencies:**

   ```bash
   npm install
   ```

2. **Start development server:**

   ```bash
   npm run dev
   ```

3. **Build for production:**

   ```bash
   npm run build
   ```

## File Structure

The project follows a logical and scalable folder structure:

- `components/`: Reusable UI components
- `pages/`: Individual page components
- `layouts/`: Overall page layouts (e.g., DashboardLayout)
- `lib/`: Utility functions, API integrations, etc.
- `assets/`: Static assets
- `types/`: Typescript Types

## Customization

- Edit components in `src/components` to fit your needs
- Update styles in `src/App.css` and `src/index.css`
- Add new pages or layouts as needed
